# 一、提前了解

## 1、基本常识了解

### 一、考试科目


共四门：

两门公共课：政治、英语;

一门基础课：数学或专业基础;

一门专业课(分为 13 大类)：哲学、经济学、法学、教育学、文学、历史学、理学、工学、农学、医学、军事学、管理学、艺术学等。

统考专业课：法硕、西医综合、教育学、历史学、心理学、计算机、农学等；非统考专业课各高校自主命题

 

### 二、考研分数


总分 500 分

政治：100 分

英语：100 分

数学或专业基础：150 分

专业课：150 分

工商管理硕士、公共管理硕士、工程硕士、旅游管理硕士、图书情报硕士、审计硕士、会计硕士初试科目为管理类联考综合和英语二，其中英语二 100 分，管理类综合 200 分。

 

### 三.报名时间(具体以网站公布为准)

 

1、中国研究生招生信息网：http://yz.chsi.com.cn

2、预报名：9 月 24-9 月 27 日每天 9：00-22：00

3、正式报名：10 月 10 日-10 月 31 日每天 9：00-22：00

4、现场确认：各省时间不一样，部分采用网上确认的方式

 

### 四、考试时间

 

基本在每年 12 月底，每科 3 个小时

第一天上午政治：8：30-11：30

下午英语：2：00-5：00

第二天上午数学或专业基础课：8：30-11：30

考试成绩公布时间：每年 2、3 月份

复试一般在 4-5 月

 

### 五、国家线、院校线、单科线

 

1、国家线：国家线即全国硕士研究生考生进入复试的基本分数要求，是基础分数线，若有资格参加复试必须过了这条线。国家线一般在每年的 3 月底 4 月初公布。

考研国家线划分为 A、B 两类，A 类分数线最高，其次是 B 类地区。国家按照一类、二类确定考生参加复试基本分数要求，一般相差 2—10 分。

一类：北京、天津、上海、江苏、浙江、福建、山东、河南、湖北、湖南、广东、河北、山西、辽宁、吉林、黑龙江、安徽、江西、重庆、四川、陕西 21 个省(市)。

二类：内蒙古、广西、海南、贵州、云南、西藏、甘肃、青海、宁夏、新疆 10 个省(区)。

如果考生成绩有一科或者总分不过国家线，那么就意味着此次考研失败，也没有机会参加调剂。但是，如果报一区的考生单科或者总分没过一区的分数线，但是过了二区国家线，就有机会参加二区招生单位的调剂。

 

2、校分数线(简称学校线)：院校线是复试资格线，国家线是录取资格线。院校线决定能否进复试，国家线决定能否读研。通过了国家线且通过院线的考生好好准备复试即可，通过了国家线未通过院线的考生需要联系调剂。院校线是各招生单位在国家线的基础上，根据本校有关专业生源余缺确定的复试资格线，一般学校直接采用了国家线。另教育部批准 34 所院校可以自主确定分数线(自划线)。

34 所自主划线学校：

北京：清华大学，北京大学，中国人民大学，北京师范大学，北京航空航天大学，北京理工大学，中国农业大学

天津：天津大学，南开大学

山东：山东大学

江苏：南京大学，东南大学

浙江：浙江大学

上海：复旦大学，上海交通大学，同济大学

广东：中山大学，华南理工大学

湖南：湖南大学，中南大学

湖北：武汉大学，华中科技大学

黑龙江：哈尔滨工业大学

吉林：吉林大学

辽宁：东北大学，大连理工大学

安徽：中国科学技术大学

陕西：西安交通大学，西北工业大学

甘肃：兰州大学

重庆：重庆大学

四川：四川大学，电子科技大学

 

3、单科线：单科线是各科的分数线，每科都过单科线，总分过国家线才有资格参加复试。

 

### 六、学硕和专硕的区别

 

1、学硕和专硕相比招生人数较少，报考难度更大。

2、学硕学制一般 3 年，学费每年 7000—8000，专硕一般 2—3 年，学费每年至少 1 万。

3、学硕以培养教学和科研人才为主，更侧重于学术；专硕是以培养应用型和工程研究型人才为目的，更侧重于实践。

4、学硕可以直博或者硕博连读，专硕想要读博须要参加博士生考试。

 

### 七、英语一和英语二的区别

 

1、学硕考英一，专硕一般考英二（个别院校除外）。

2、英一的难度要高于英二

①阅读理解：英二文章难度稍低于英一，选项比英一简单一些。

②新题型：英一主要是七选五，选小标题和排序；英二是选小标题对应、多项对应，英二的新题型比较简单，是得分的重点。

③翻译：英一是对一篇文章中的五个画线句子进行翻译；英语二是全篇翻译，比较简单，是得分的重点。

④作文：英语一的小作文是以写信为主，大作文以图画为主；英语二的大作文是以图表为主。英语二大作文的写作字数少于英语一,英一 160—200 字，英二 150 字；英二分数低于英一，英一 20 分，英二 15 分。

 

### 八、不考数学的专业

 

艺术学、管理类专业（行政管理、土地资源管理、社会管理）、历史学、文学、医药类专业、哲学、教育学、法学

 

### 九、数一、数二和数三的区别

 

【考数一的专业】①工学门类的力学、机械工程、光学工程、仪器科学与技术、冶金工程、动力工程及工程热物理、电气工程、电子科学与技术、信息与通信工程、控制科学与工程、计算机科学与技术、土木工程、水利工程、测绘科学与技术、交通运输工程、船舶与海洋工程、航空宇航科学与技术、兵器科学与技术、核科学与技术、生物医学工程等一级学科中所有的二级学科、专业以及管理学门类中的管理科学与工程一级学科中所有的二级学科、专业。

【考数二的专业】工学门类的纺织科学与工程、轻工技术与工程、农业工程、林业工程、食品科学与工程等一级学科中所有的二级学科、专业

【考数三的专业】经济类和管理类的为数学三，经济类和管理类包括经济学类的各一级学科、管理学类中的工商管理、农业经济管理的一级学科和授予管理学学位的管理科学与工程的一级学科。

 

### 十、复试与调剂

 

一般考研复试学校会需要 2-3 天，有的院系一天就可以完成，具体看学校安排

复试分为笔试和面试（有的学校专业只有面试）

其中笔试分为英语的听力笔试和专业笔试，面试分为英语口语和专业课面试

复试比例一般在 120%-150%

复试在最终录取分数中的占比一般不超过 50%

 

分数刚过院系线需要做两手准备：一、认真准备学校复试，有逆袭的可能，二、时刻关注调剂信息，做好调剂的准备。

 

调入专业与第一志愿报考专业相同或者相近，一般专业代码前四位相同即可互相调剂。专业课可以不相同，但是统考科目一般情况下必须相同。初试有数学可以调剂到初试没有数学的专业，反之不可，数学考试分数一、数二、数三，难度依次递减，考难的可以调剂到简单的，反之不可。英语分为英一和英二，英一难度大于英二，考英一的可以调剂到考英二的。

 

考研调剂基本的流程如下：1、高校先发布调剂信息 2、考生填写调剂申请 3、考生给接受调剂单位寄申请材料 4、接收调剂的单位给符合条件的考生寄接收调剂函 5、考生向原报考单位申请调剂 6、原报考单位转寄报考材料给接受调剂单位 7、接收调剂单位接收考生调剂资料 8、调剂考生参加复试 9、被录取或者重新联系

考研调剂需要从网上登录调剂系统，调剂系统要求一次只能填两所院校，并且会在 48 小时之内锁定，48 小时之后才能更改。

### (1)专硕与学硕的区别

|  | 专硕 | 学硕 |
| :-- | :--: | :--: |
| 招生对象 | 除个别专业应届生都可报 | 在职及应届  |

|  | 全日制研究生 | 非全日制研究生 |
| :-- | :--: | :--: |
| 录取分数/难度 | 高.学硕很难 | 较低.但名校不容易  |

| 报考门槛 | 部分专硕有工作经历限制 | 允许应届生报考  |
| 学费情况 | 学硕低.但部分专硕高 | 一般能承受.部分较夸张 |
| 助学贷款 | 可以申请 | 不能申请 |
| 档案户口 | 可以转.但部分专硕有限制 | 原则上不可以转 |
| 毕业证书 | 双诳.会注明全日制 | 双证.注明非全日制 |

| 期刊论文 | 学硕会有眼球.名校要求高 | 一般不要求.和专硕类似 |
| 就业歧视 | 专硕学硕无差别 | 有歧视现象.含金量不如全日制.但总比本科强 |
| 能否读博 | 可以.学硕更有优势 | 可以.有成果最好 |
| 导师规格 | 完全一致 | 完全一致 |
| 整体含金量 | 现在高.未来会趋同 | 现在低.未来发展是大趋势 |
| 未来趋势预测 | 专硕性价比最高.学硕招生越来越少.难度越来越大 | 报考限制会增多.录取难度会增大.尤其是名校专业 |

### (2)具体重要时间点（共18个）

> 拿到考试大纲！！！！！！

**<1>7月下旬-8月中旬**

教育部定制2018年全国研究生招生计划。全国各研招机构根据教育部2017招生的要求和本机构上一年度的招生计划完成情况，上报2018年度研究生招生计划(含保送生名额)。教育部汇总各研招机构计划，制定全国2018年硕士研究生招生计划。

提醒：这个阶段正好是大家放暑假的时候。每年都有很多同学为了备考而选择留在学校参加辅导班学习，也有不少同学依然如故回家过暑假。不管怎么说，要记住，暑假的两个月对考研很重要！这是极少有的一段完整的没有学校上课干扰的复习时间，它决定着在9-10月这个强化阶段自身的学习水平和能力是否有所提升。

**<2>8月下旬-9月初**

全国招生简章正式公布，各研招机构根据2018年全国硕士研究生入学考试招生简章要求公布本机构2018年硕士研究生招生简章及招生专业目录。

提醒：在备考阶段切记不要每天只是一味的学习。一定要用科学、灵活的用功方式。死读书是会读死书的。考研备考很需要和各个阶段、各个环节的同学沟通。要时刻保证自己没有偏离正确方向。

**<3>9月中上旬**

2018年硕士研究生入学考试大纲正式发行。政治、英语及部分专业课考试大纲及大纲解析陆续出版发行。注意：这只是一般参考时间，具体每年会有一些变动。

提醒：每年都有同学因为不重视考试大纲而吃亏，有些同学认为只要学好自己手头的公共课和专业课书籍就可以了。其实则不然，考试大纲及大纲解析是对考研整体范围的一个划定和考试内容的解释说明，吃透考试大纲，就可以省很多力，对于大纲不要求的范围，同学们可以相对的放松。

**<4>9月中旬**

全国硕士研究生入学考试网上预报名，各大高校应届大四毕业生可以在网上进行预报名。

提醒：网上报名看起来简单，可实际操作起来却有不少专业性的疑问。每一项信息该如何填写，保证填写正确无误才是最关键的。比如考生类别、考试方式、户口、档案所在地等等这些重要信息的填写，建议大家在填写前上网搜索一下相关内容，咨询一下师兄师姐或者相关老师。落实清楚了再填写，不要着急往上写。

**<5>10月中旬**

全国硕士研究生入学考试网上正式报名。

提醒：这一阶段考生需要关注的事情是比较多的。如硕士研究生考生一律采取网上报名方式，逾期不再补报，也不得再修改报名信息。请大家按教育部、考生所在地省级高校招生办公室、报考点以及报考招生单位的网上公告要求报名。凡不按公告要求报名、网报信息误填、错填或填报虚假信息而造成不能考试或复试的后果，由考生本人承担。大家一定不要忘了报名，不然白白付出了这么多努力。

**<6>11月中旬**

全国硕士研究生入学考试报名信息现场确认。（有些考点是需要现场确认，有些是在线确认学历信息，不用到现场确认，这个需要大家报名之后随时关注考点公布的信息。）

提醒：大家需要到本人所在的省、自治区、直辖市高校招生办公室公告指定的报考点确认;考生持本人身份证、学历证书(普通高校和成人高校应届本科毕业生持学生证)和网上报名编号，由报考点工作人员核对;考生确认本人网报信息；考生按规定缴纳报考费(考生办理报考手续缴纳报考费后，不再退还)；到报考点按规定采集考生本人图像信息(照相)。

**<7>11月下旬**

全国硕士研究生入学考试非统考专业课命题工作正式开展，各个研招单位自主命题制该硕士生招生点的专业课试题。

提醒：保持好自己的学习状态，不要被其他国考、找工作什么的干扰。

**<8>12月上旬**

准考证开始可以下载打印。

提醒：关注打印准考证等事项，一定要和同学小伙伴保持联系，互相分享最新消息。

**<9>12月底**

每年考研初试时间在12月底，17考研初试时间是2016年12月24日开始，大家可以参考。该阶段大家需要注意调整作息时间、平和心态、全力以赴准备初试。

提醒：这里要告诫大家，考试前一天千万不要再熬夜看书了。考研对于每位考生来说都是复习不完的，因为当你复习的越多，知识面越广，就会发现自己未知的东西也越多。考前你需要的是自信，相信自己之前的努力是不会白费的。每年都有不少很「勤奋」的同学，因为考试前一天熬夜背题，结果没有休息好，第二天考试状态极差，导致连正常水平都没有发挥出来。

**<10>考研初试后**

全国硕士生入学考试阅卷开始，北京地区首先开始阅卷，教育部考试中心根据北京地区阅卷情况制定阅卷标准全国范围内进行阅卷。

提醒：大家可以根据自己考试的情况，开始准备复试，包括询问学长学姐，查询复试的相关信息，咨询辅导机构，或是联系报考院校询问复试信息等。

**<11>次年春季**

各院校将公布考生分数，大家可以通过热线电话、网络等途径查询自己的考分。

提醒：：查分后，如有分数上的疑问、立刻申请分数信息核对。根据周围同学的分数，和今年总体平均分数预测目标院校目标专业的分数线。有望进入复试阶段的童鞋一定要抓紧时间多为复试做准备，觉得分数危险的可以查查调剂院校的信息，以防万一。

**<12>次年2月中旬**

34所自定义分数线的研究生招生单位陆续公布自己的复试分数线。

提醒：大家需要及时核实自己是否通过复试分数线准备复试或者调剂。

**<13>次年3月(预计时间)**

国家复试分数线公布。国家按照13大学科门类，对一类、二类地区以及单独考试和特殊专业考试进行复试分数线划分。

提醒：要密切关注分数线的公布情况核实自己是否通过复试分数线。

**<14>次年3月中下旬**

全国各研招单位复试工作陆续进行。

**<15>次年3月中下旬(国家线公布之后)**

调剂工作开始。

招生单位及时发布生源余缺信息，符合教育部调剂规定的考生，可在规定时间内通过「中国研究生招生信息网」登录调剂志愿，招生单位选出参加复试的考生名单，并通知考生复试，招生单位在规定时间内组织复试，并将复试结果通知考生，接收调剂生的招生单位向第一志愿单位发函调考试档案，做最后审核，确定是否录取。

**<16>次年3月至4月（根据学校情况，基本和复试一起）**

体检，考生复试时应按招生单位规定到指定的二级甲等以上(含二级甲等)医院进行体格检查。

提醒：：加强锻炼、保持身体健康，确保顺利通过体检。研究生入学体检和高考时的体检类似，从体检项目和要求上基本一致。

**<17>次年4月到5月（根据学校情况）**

政审招生院校对档案审查，包括组织关系。同时要求填写政审表格，保管档案的单位、学校、人才中心或者居委会的盖章。

提醒：：及时办理所要求事项，态度上要积极主动。

**<18>次年4月至5月（根据学校情况）**

招生单位根据国家下达的招生计划，考生入学考试的成绩(含初试和复试)并结合其平时学习成绩和思想政治表现、业务素质以及身体健康状况确定录取名单。
